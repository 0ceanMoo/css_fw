水平メニューは基礎設定のみ提供する。細かい設定は「ul._virticaled」を継承して作るか「ul._virticaled」に属性を追加して作るかすること

sassが使えるなら

.myList {
  @extend ._virticaled
  li {
    background: #ccF;
    width: 100px;
    height: 60px;
    line-height: 60px;
    text-align: center;
  }
}

styleタグ

ul._virticaled > li {
  background: #ccF;
  width: 100px;
  height: 60px;
  line-height: 60px;
  text-align: center;
}

カードの説明

.myCard {
  @extend ._Card-base
  .image
    // 設定
  .title
    // 設定
  .content
    // 設定
}

カードデザインについてはいくらデザイン作っても、実際に使う時どうせそのまま使えないので「._Card-base」という基本設定を持つクラスを@extendして、細かい部分は自前でカスタマイズすること。
