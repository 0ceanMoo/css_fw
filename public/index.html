水平メニューは基礎設定のみ提供する。細かい設定は「ul.--virticaled」にstyleを追加するか、sassが使えるなら「ul.my.--virticaled」を継承して整える

styleタグ

/* ul.--virticaledに直接styleあてると、他にも影響するので「my」で影響範囲を絞ってる */
ul.my.--virticaled > li {
  background: #ccF;
  width: 100px;
  height: 60px;
  line-height: 60px;
  text-align: center;
}

sassが使えるなら

.myList {
  @extend .--virticaled
  li {
    background: #ccF;
    width: 200px;
    height: 60px;
    line-height: 60px;
    text-align: center;
  }
}

カードの説明

div.my._Card {
  width: 300px;
  margin-right: 1rem;
}
div.my._Card > * {
  padding: .7rem 1rem;
}
div.my._Card > h4 {
  background: black;
  color: white;
}
div.my._Card > p {
  color: #666;
  font-size: .9rem
}

カードデザインについてはいくらデザイン作っても、実際に使う時どうせそのまま使えないので「._Card」という基本設定を持つクラスを@extendして、細かい部分は自前でカスタマイズすること。

カードの説明

.myCard {
  @extend ._Card
  .image
    // 設定
  .title
    // 設定
  .content
    // 設定
}
